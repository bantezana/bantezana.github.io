Clever Cleats
Internet of Things

Tracking the force and type of kick the user takes in competitive and progressive application for all skill levels. Help players, coaches and care takers

First in thinking of coding on how to compile from arduino, we needed its serial port(/dev/cu.usbmodem1421). From the start we import Application Program Interfaces such as serial and matplotlib in Python

With the help of an Biomedical graduate, we were able to capture and depict the amount of force(in grams) that the player gives through each kick of the ball. We used Arduino to take in the data shown on the figure above while the bluetooth module takes in the data and transmits to the mobile appliaction I created in order to view the data on hand. We also used a PCB to configure the connections from the sensor to the Arduino.
